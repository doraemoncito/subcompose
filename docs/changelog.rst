Changelog
=========

0.1.0 (unreleased)
------------------

* Initial release.
* Modular package structure (``src/`` layout).
* Support for ``x-subcompose-groups`` and ``x-subcompose-managed`` extension
  fields.
* Commands: ``run``, ``stop``, ``preview``, ``validate``, ``pull``, ``push``,
  ``tag``, ``delete-containers``, ``delete-images``.
* ``--unmanaged`` flag to exclude managed services and their dependants.
* Shell-style variable interpolation via ``--interpolate``.
